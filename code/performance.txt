File Paths
- VScode starts path from main dir (Space Shooter), while Sublime starts from the python file.
- Use os.path.join() for correct path separator

Surfaces 
- Use convert() or convert_alpha() on image surfaces
- Not just the display screen!!! You can draw and blit on any individual surfaces (Suitable for sprites)
- Tip: Treat a newly created surface as a blank canvas (Add flag pygame.SRCALPHA for transparency) 

Masks
-  To increase performance, create and set a mask attribute for all sprites that will use this 
function to check for collisions. Otherwise, each time this function is called it will create new masks.

Miscellaneous
- Close the game with pygame.quit()
